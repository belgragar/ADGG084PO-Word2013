# 🌱 EJERCICIOS DE REFUERZO PARA ALUMNAS CON CERO NIVEL  
## (Con plantillas imprimibles)

# 🌼 **DÍA 1 – Primer contacto con Word y tablas muy básicas**

## 🧩 **Ejercicio de refuerzo**
1. Abrir Word  
2. Escribir tres líneas de texto  
3. Insertar una tabla de 2 columnas y 2 filas  
4. Escribir dentro de cada celda  
5. Guardar el documento  

### 📄 **Plantilla imprimible**

```
EJERCICIO DE REFUERZO – DÍA 1

1. Escribe tu nombre:
__________________________________________

2. Escribe estas tres frases:
- Estoy aprendiendo Word.
- Hoy he creado mi primera tabla.
- Me siento más segura usando el ordenador.

3. Inserta una tabla de 2x2 y escribe dentro:

┌───────────────┬───────────────┐
│ Planta         │ Precio (€)    │
├───────────────┼───────────────┤
│                │               │
└───────────────┴───────────────┘

4. Guarda el documento con el nombre:
“Ejercicio_Dia1_TuNombre”
```

---

# 🌼 **DÍA 2 – Insertar imágenes y crear columnas simples**

## 🧩 **Ejercicio de refuerzo**
1. Escribir un título  
2. Insertar una imagen  
3. Crear dos columnas  
4. Escribir un texto corto en cada columna  

### 📄 **Plantilla imprimible**

```
EJERCICIO DE REFUERZO – DÍA 2

1. Escribe este título:
PLANTAS QUE ME GUSTAN

2. Inserta una imagen de una planta.

[Espacio para imagen]

3. Crea dos columnas.

4. En la columna izquierda escribe:
Plantas de huerta:
- Lechuga
- Tomate
- Pimiento

5. En la columna derecha escribe:
Plantas aromáticas:
- Menta
- Romero
- Lavanda
```

---

# 🌼 **DÍA 3 – Crear un gráfico muy sencillo**

## 🧩 **Ejercicio de refuerzo**
1. Crear una tabla con 3 plantas y cantidades  
2. Insertar un gráfico de columnas  
3. Guardar el documento  

### 📄 **Plantilla imprimible**

```
EJERCICIO DE REFUERZO – DÍA 3

1. Crea esta tabla:

┌───────────────┬───────────────┐
│ Planta         │ Cantidad      │
├───────────────┼───────────────┤
│ Lechuga        │      5        │
│ Tomate         │      3        │
│ Pimiento       │      2        │
└───────────────┴───────────────┘

2. Inserta un gráfico de columnas con estos datos.

3. Guarda el documento como:
“Grafico_Basico_TuNombre”
```

# 🌼 **DÍA 4 – Títulos y navegación básica**

## 🧩 **Ejercicio de refuerzo**
1. Escribir tres títulos usando “Título 1”  
2. Escribir un pequeño texto debajo  
3. Insertar una tabla de contenido  

### 📄 **Plantilla imprimible**

```
EJERCICIO DE REFUERZO – DÍA 4

Escribe estos títulos usando “Título 1”:

TÍTULO 1: MI HUERTA
Texto:
__________________________________________

TÍTULO 1: MIS FLORES
Texto:
__________________________________________

TÍTULO 1: MIS ARBUSTOS
Texto:
__________________________________________

Inserta una tabla de contenido al principio del documento.
```

# 🌼 **DÍA 5 – Formularios muy simples**

## 🧩 **Ejercicio de refuerzo**
1. Insertar un cuadro de texto  
2. Insertar una casilla de verificación  
3. Insertar un selector de fecha  

### 📄 **Plantilla imprimible**

```
EJERCICIO DE REFUERZO – DÍA 5

Crea este formulario básico:

Nombre:
[Cuadro de texto]

Fecha:
[Selector de fecha]

Tipo de planta:
▢ Huerta
▢ Aromática
▢ Arbusto
```

# 🌼 **DÍA 6 – Revisión muy básica**

## 🧩 **Ejercicio de refuerzo**
1. Activar control de cambios  
2. Modificar una frase  
3. Insertar un comentario  

### 📄 **Plantilla imprimible**

```
EJERCICIO DE REFUERZO – DÍA 6

1. Activa el control de cambios.

2. Modifica esta frase:
“Las plantas estarán disponibles en junio.”

3. Inserta un comentario que diga:
“¿Seguro que es junio?”
```

---

# 🌼 **DÍA 7 – Correspondencia muy básica**

## 🧩 **Ejercicio de refuerzo**
1. Crear un documento principal  
2. Crear un origen de datos con solo un campo: Nombre  
3. Insertar el campo combinado  

### 📄 **Plantilla imprimible**

```
EJERCICIO DE REFUERZO – DÍA 7

1. Crea un origen de datos con un solo campo:
Nombre

2. Inserta este texto:

Hola «Nombre»:
Gracias por visitar el vivero.

3. Combina el documento.
```

---

# 🌼 **DÍA 8 – Hipervínculos básicos**

## 🧩 **Ejercicio de refuerzo**
1. Crear un título  
2. Insertar un enlace a una web  
3. Crear un marcador  
4. Insertar un enlace interno  

### 📄 **Plantilla imprimible**

```
EJERCICIO DE REFUERZO – DÍA 8

1. Escribe este título:
INFORMACIÓN DEL VIVERO

2. Inserta un enlace a:
https://www.tineo.es

3. Crea un marcador llamado “contacto”.

4. Escribe al final del documento:
SECCIÓN DE CONTACTO

5. Inserta un enlace interno que lleve al marcador “contacto”.
```
