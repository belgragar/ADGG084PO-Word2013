# 🌼 **DÍA 1 – Tablas básicas**

## 🧩 **Ejercicio de ampliación (nivel básico)**  
Crear una **tabla sencilla** con información de 3 plantas del vivero:

- Nombre  
- Tipo  
- Precio  

Sin imágenes, sin anidar tablas, sin autoformato complejo.

### 📄 **Plantilla imprimible**

```
TABLA BÁSICA – PLANTAS DEL VIVERO

┌───────────────────────┬───────────────────────┬───────────────┐
│ Nombre de la planta   │ Tipo                  │ Precio (€)     │
├───────────────────────┼───────────────────────┼───────────────┤
│                       │                       │               │
├───────────────────────┼───────────────────────┼───────────────┤
│                       │                       │               │
├───────────────────────┼───────────────────────┼───────────────┤
│                       │                       │               │
└───────────────────────┴───────────────────────┴───────────────┘
```

- Un título

- Una imagen pequeña insertada correctamente  

### 📄 **Plantilla imprimible**

```

“Plantas de Primavera del Vivero”

COLUMNA 1 (texto)
───────────────────────────────────────────────
_______________________________________________
_______________________________________________

───────────────────────────────────────────────
COLUMNA 2 (imagen)
───────────────────────────────────────────────
[Insertar imagen pequeña]
```

# 🌼 **DÍA 3 – Gráfico sencillo**

# 🌼 **DÍA 6 – Revisión básica**

## 🧩 **Ejercicio de ampliación (nivel básico)**  
Practicar:

- Activar control de cambios  
- Escribir un texto  
- Insertar un comentario  

### 📄 **Plantilla imprimible**

```
EJERCICIO DE REVISIÓN – NIVEL BÁSICO

Texto para modificar:
“Las plantas aromáticas estarán disponibles a partir de mayo.”

Comentario sugerido:
“Cambiar ‘mayo’ por ‘abril’ si llega antes el pedido.”
```

# 🌼 **DÍA 7 – Correspondencia básica**

## 🧩 **Ejercicio de ampliación (nivel básico)**  
Crear una carta con **un solo campo combinado**: el nombre.

### 📄 **Plantilla imprimible**

```
CARTA BÁSICA – COMBINACIÓN DE CORRESPONDENCIA

Estimado/a «Nombre»:

Le informamos de que su pedido está listo para recoger.

Atentamente,
Vivero “La Flor del Narcea”
```

# 🌼 **DÍA 8 – Hipervínculos básicos**

```
_______________________________________________

_______________________________________________
```
