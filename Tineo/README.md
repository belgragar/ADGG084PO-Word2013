# 📘 **MANUAL PEDAGÓGICO DE MICROSOFT WORD**  
### *Guía completa de aprendizaje – Nivel Básico, Intermedio, Avanzado y Refuerzo*

---

# 🧭 **INTRODUCCIÓN GENERAL**

Este manual pedagógico está diseñado para acompañar el aprendizaje progresivo de Microsoft Word, desde los fundamentos hasta las herramientas profesionales.  

Incluye:
- Explicaciones conceptuales  
- Ejemplos guiados  
- Actividades prácticas  
- Ejercicios de refuerzo  
- Plantillas completas  
- Preguntas de reflexión  
- Recomendaciones didácticas  

Está organizado en **cuatro módulos**, cada uno con ocho sesiones (días), pensadas para trabajar de forma secuencial o independiente.

---

# 🎯 **OBJETIVOS GENERALES DEL MANUAL**

Al finalizar este manual, el estudiante será capaz de:

### Competencias técnicas
- Crear documentos profesionales con estructura clara.  
- Aplicar formato coherente a texto, listas, tablas e imágenes.  
- Utilizar estilos, encabezados, pies y secciones.  
- Generar índices automáticos y documentos maestros.  
- Insertar gráficos, tablas dinámicas y elementos visuales.  
- Automatizar tareas mediante macros.  
- Crear correspondencia simple y condicional.  
- Integrar contenido HTML, CSS y herramientas de IA.

### Competencias transversales
- Organizar información de forma clara y lógica.  
- Revisar y mejorar documentos con criterio profesional.  
- Desarrollar autonomía en el uso de herramientas digitales.  
- Aplicar pensamiento crítico en la presentación de información.

---

# 🧩 **METODOLOGÍA PEDAGÓGICA**

Este manual sigue una metodología basada en:

### ✔ Aprendizaje progresivo  
Cada módulo incrementa la complejidad de forma gradual.

### ✔ Aprendizaje activo  
El estudiante realiza ejercicios prácticos en cada sesión.

### ✔ Aprendizaje significativo  
Los ejercicios están basados en documentos reales: informes, folletos, correspondencia, etc.

### ✔ Evaluación continua  
Cada día incluye preguntas de reflexión para consolidar el aprendizaje.

### ✔ Transferencia  
El módulo de refuerzo integra todas las habilidades en documentos completos.
