Aquí tienes **los archivos base del vivero en versión *básica***, pensados específicamente para **alumnas con cero nivel**, que necesitan documentos muy simples, limpios y sin elementos que puedan confundirlas.

Estos archivos están diseñados para:

- Ser extremadamente fáciles de usar  
- No contener estilos complejos  
- No incluir tablas complicadas  
- No requerir conocimientos previos  
- Permitir que las alumnas practiquen acciones básicas: escribir, insertar, guardar, seleccionar, etc.  

Están listos para **copiar y pegar en Word** tal cual.

---
