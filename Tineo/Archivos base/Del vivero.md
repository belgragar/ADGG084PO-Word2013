Aquí tienes **los archivos base del vivero**, listos para copiar en Word y usarlos como **material inicial para todos los ejercicios del curso**.

Son documentos “semilla” que las alumnas podrán abrir, modificar y completar durante las sesiones.  
Están diseñados para que funcionen como **punto de partida** para:

- Tablas  
- Catálogos  
- Fichas  
- Formularios  
- Correspondencia  
- Datos para gráficos  
- HTML  
- Macros  
- Documentos maestros  

Todo está en **Markdown**, pero pensado para pegar directamente en Word.
