# **1. Introducción y Justificación**

Este curso busca que las alumnas adquieran competencias avanzadas en Microsoft Word 2013 mediante un enfoque práctico, contextualizado y progresivo.  
El **eje central del curso** será un **ejercicio guiado continuo**: la creación, gestión y preparación de documentación para un **vivero rural** que se dispone a iniciar la campaña de venta de plantas de huerta, arbustos, frutales, aromáticas y medicinales.

Este ejercicio servirá como hilo conductor para explicar todos los contenidos avanzados del curso.

Además, se propondrán **ejercicios alternativos** relacionados con otros sectores del mundo rural:  
- Ganadería  
- Gestión de fincas  
- Elaboración de productos artesanales (miel, queso, mermeladas…)  
- Turismo rural, gastronómico y de naturaleza  
- Artesanía y oficios tradicionales  

---

# **2. Objetivos**

## **2.1. Objetivos Generales**
- Dominar herramientas avanzadas de Word 2013 para crear documentos profesionales y complejos.
- Integrar tablas, gráficos, organigramas, formularios y elementos multimedia.
- Gestionar documentos extensos mediante referencias, documentos maestros y control de cambios.
- Automatizar tareas mediante macros.
- Realizar combinaciones de correspondencia y trabajar con datos externos.
- Conocer opciones de seguridad y distribución de documentos.

## **2.2. Objetivos Específicos**
- Crear y modificar tablas complejas aplicadas al inventario del vivero.
- Elaborar gráficos y diagramas basados en datos de producción y ventas.
- Diseñar documentos estructurados para catálogos, fichas y manuales.
- Crear formularios para pedidos, fichas de plantas y control de stock.
- Realizar combinaciones de correspondencia para clientes del vivero.
- Incrustar y vincular datos desde Excel.
- Proteger documentos y gestionar versiones.
- Exportar documentos a HTML e insertar hipervínculos.

---

# **3. Metodología**

## **3.1. Enfoque metodológico**
- **Ejercicio guiado único y continuo**:  
  Cada contenido se explicará aplicándolo al documento del vivero, que irá creciendo y volviéndose más complejo.
- **Aprendizaje práctico**:  
  Tras la explicación, las alumnas replican el ejercicio y realizan una ampliación.
- **Ejercicios alternativos**:  
  Actividades relacionadas con otros sectores rurales para reforzar o ampliar.
- **Atención a la diversidad**:  
  - Refuerzo para alumnas con menor base.  
  - Ampliación para alumnas avanzadas.  
- **Evaluación continua**: mini test diario.  
- **Evaluación final**: examen tipo test + exploración de herramientas de IA aplicadas a Word.
